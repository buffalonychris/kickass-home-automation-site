How much does it cost?

Budgets vary by home size and goals. These ranges are realistic starting points.

Starter (1–2 rooms)

  • Lighting + presence
  • Leak sensor + notifications
  • Basic voice control

Typical: $1.5k–$4k

Core (Whole main floor)

  • Lighting scenes + shades
  • Thermostat + schedules
  • Washer/dryer done alerts

Typical: $5k–$10k

Full‑stack

  • Z‑Wave/Zigbee mesh
  • Security sensors + camera tie‑ins
  • Local control; cloud optional

Typical: $12k+

Hardware costs are estimates; final scope is verified with photos and a walk‑through.
